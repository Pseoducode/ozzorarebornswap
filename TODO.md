## Steps untuk Menjalankan App di Localhost

### Step 1: Install Dependencies

- Jalankan `npm install` untuk menginstall dependencies

### Step 2: Configure .env

- Buat file `.env` dengan variabel yang diperlukan:
  - RPC_URL (Polygon RPC)
  - OLD_TOKEN_ADDRESS
  - NEW_TOKEN_ADDRESS
  - OLD_TOKEN_RECEIVER
  - PLATFORM_FEE_RECEIVER
  - TREASURY_PRIVATE_KEY

### Step 3: Start Server

- Jalankan `npm start` atau `node server.js`

### Step 4: Akses Frontend

- Buka browser ke <http://localhost:8787>
